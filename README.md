# Rice Hyprland
Customização **neo-retro** do compositor Wayland **Hyprland**, com estética simples e
workflow focado em produtividade através de atalhos de teclado.

Projeto pessoal, pensado para uso diário.

---

## Dependências:
- hyprland
- hyprlock
- hyprpaper
- nemo
- waybar
- wofi
- kitty
- firefox-dev
- mako
- fastfetch
- pipes
- cmatrix
- cbonsai
- cava
- tty-clock
- git

---

## Instalação:
- É de grande importância deixar claro que utilizo o Fedora e que este rice foi pensado para o meu uso. 
  Então, é provável que não seja 100% compatível com todas as máquinas e sistemas logo de cara.
- minha resolução é 1920x1080, se a sua for diferente os aplicativos ficarão fora do enquadramento.

```bash
mkdir ~/.config/Hypr_rice-th
git clone https://github.com/THmaguetas/my-rice-hyprland ~/.config/Hypr_rice-th
cd ~/.config/Hypr_rice-th
chmod +x install_script.sh
./install_script.sh
```
